Chess AI Project

Building a smarter way to play chess with AI-powered move analysis, real-time strategies, and game recommendations.

Learn More
Project Overview

This Chess AI project aims to help players enhance their chess skills. It includes real-time move recommendations, statistical analysis of game states, and a user-friendly interface. Powered by cutting-edge machine learning and integrated into a web app using Streamlit, it makes chess strategy accessible to all.

Key Features
  • Real-time move recommendations
  • Custom scoring system for move evaluation
  • Interactive chessboard for gameplay
  • Streamlit integration for web app accessibility
Technologies Used
  • Python
  • TensorFlow
  • Streamlit
  • Scikit-learn
